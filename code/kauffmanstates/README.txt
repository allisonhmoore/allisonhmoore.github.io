How to run the program kauffmanstates.py

Download kauffmantstates.py to a local directory

Open a terminal window, and navigate to the local directory containing the program kauffmanstates.py

In the terminal window, type 
>> python kauffmanstates.py

The default input is for the figure eight knot. To run the program for any other knot, modify the input in lines 11-16 of the file kauffmanstates.py with a text editor.

Detailed instructions for producing correct input can be found in the file combinatorialalexanderpolynomial.pdf.
